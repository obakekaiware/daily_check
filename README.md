## Usage

ファイル「data/urls.json」に登録したいWebページのRSSフィードを入力する。

## Installation

次のコマンドで必要ライブラリを取得する。

```shell
pip install -r requirements.txt
```

## Run

```shell
streamlit run main.py
```
